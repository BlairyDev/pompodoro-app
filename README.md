# About MVVM

Model - View - ViewModel

Model - data layer, business logic, or this is where network api calls is

ViewModel - The bridge between Model and View. When View needs to Model it needs to communicate to ViewModel and this is where state lives

View - this is the UI Layer(Jetpack Compose) this only have UI nothing else


For more details: https://developer.android.com/topic/architecture
